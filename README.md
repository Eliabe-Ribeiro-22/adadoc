# ADADOC

## Sobre o projeto

Sistema de Gestão de alunos ADADOC. 

O sistema cadastrará alunos. Além disso, ele fará chamada e lançará notas para os mesmos.

Também será feito um sistema de login para somente os professores acessarem o sistema de gestão

## Tecnologias utilizadas

Laravel, JS, CSS, HTML, PHP e MySQL
